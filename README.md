# Sitio Web - Portafolio Personal

## Autor

Guido Fermani

## Descripción

Este proyecto consiste en un sitio web de portafolio personal desarrollado como trabajo práctico del curso de Desarrollo Web.

El sitio fue construido utilizando:

- HTML5
- CSS3
- Bootstrap 5
- Flexbox
- CSS Grid
- Git
- GitHub

## Páginas

- Inicio
- Sobre mí
- Proyectos
- Habilidades
- Contacto

## Características

- Diseño responsive
- Navbar de Bootstrap
- Grid y Flexbox
- CSS personalizado
- Organización semántica
- Navegación entre páginas
